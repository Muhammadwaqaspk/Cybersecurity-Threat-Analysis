# Cybersecurity-Threat-Analysis
Scans network packets for Port Scanning, DDoS attacks, Malicious payloads (SQL injection, malware commands), and unauthorized port access 
<br>
Classifies threats by severity: CRITICAL / HIGH / MEDIUM / LOW 
<br>
Generates actionable security reports with blocking recommendations
<br>
<br>
Features:
<br>
Port Scan Detection - Multiple ports access check
<br>
DDoS Detection - High frequency request analysis
<br>
Malicious Payload Detection - Pattern matching (SQL injection, command injection)
<br>
Suspicious Port Monitoring - Known malicious ports (22, 445, 4444, etc.)
<br>
Threat Classification - CRITICAL, HIGH, MEDIUM, LOW levels
<br>
Detailed Report Generation - Actionable recommendations
<br>
<br>
Using:
<br>
Enterprise firewalls - Real-time intrusion detection
<br>
Security Operations Centers (SOC) - Automated threat monitoring
<br>
Cloud servers - Protect against brute-force & malware
<br>
Banks/Financial systems - Prevent data breaches
<br>
Government networks - Critical infrastructure protection
